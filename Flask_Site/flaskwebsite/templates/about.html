About This Website

This website provides information regarding prices about popular foreign exchange pairs. In the world of foreign exchange, or forex, the market of exchanging currency is one of the biggest in the world. As many people make decisions in exchanging certain forex pairs, it is important to view the prices, known as the ask and bid prices, of the pairs that the person wants to exchange.

Any person using this website can use the tool located in the home page to view the information of popular forex pairs. This will redirect the user to a new page that will generate a table that shows information of the selected pair with the following:

  • Pair Name
  • Ask Price
  • Bid Price
  • High Price (Daily High)
  • Low Price (Daily Low)
  • Percent Change (From previous high)
  • High/Low Average (Average taken from high and low prices)
    •

    This Project uses Flask, a micro-web framework used for the development of this website using Python for creating the scripts used for flask to run the website, HTML/CSS to create the layout and appearance of the website and the python modules "beatifulsoup4" and "requests" to scrape the forex information from another website's HTML page via investing.com.

    Forex information provided by: www.investing.com/currencies/streaming-forex-rates-majors
